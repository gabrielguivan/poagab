# @gabrielguivan manifesto

Landing page minimalista feita para funcionar como link de perfil, com foco em uma mensagem curta e visual:

`porto alegre soon`

## Rodar localmente

1. Garanta que voce tenha Node.js 18+.
2. Execute:

```bash
npm run dev
```

3. Abra `http://localhost:8080`.

## Arquivos principais

- `public/index.html`: estrutura da landing.
- `public/styles.css`: identidade visual editorial e responsiva.
- `public/app.js`: detalhe de interacao leve com relogio de Porto Alegre e glow seguindo o cursor.
- `server.js`: servidor estatico simples para servir a pagina.
